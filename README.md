WebAlfa IRNIC SubReseller Registrar WHMCS Module
=====================================
WHMCS registrar module for IR domains operations.

Compatible with WHMCS v7 and later

:: This module is base on WebAlfa API v2.

**Functionalities:**
- Register
- Rnew
- Transfer
- Getting nameservers
- Saving nameservers
